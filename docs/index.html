freeHorn

Composed by Larry Polansky of Frog Peak Music

JSyn Programming by Phil Burk, www.softsynth.com

for Kirsten Barrow

Operation

See specification by Larry Polansky for details of various parameters.

  1. Double click on "FreeHorn" icon. Wait ten seconds for application to appear.
  2. Set parameters by typing numbers into text fields then hitting Return or Enter key to enter the value. Field will turn red until value is entered. If the value is clipped then the field will turn yellow.
  3. Use File "Save As..." item to save parameters for the next time the program is run.
  4. Hit "Start" button.
  5. Listen to the piece.

Release Notes

April 13th, V0.4, build 17, 2004

  • Added timeAdvance so that notes are scheduled in advance and timing is more stable.

April 12th, V0.3, build 16, 2004

  • Fixed bug in voice allocation that caused notes to get cut off.

April 12th, V0.2, build 15, 2004

  • Added attackMin, attackMac, decayMin and decayMac parameters.
  • Added Pitch Scaling which scales duration, rest, attack and decay times based on harmonic index. Equation is shown in GUI choice box.
  • Rest Probability now uses 1.0 as all rests instead of all notes.
  • Improved envelope control for WaveShaper
  • Raised limit for Loudness

April 5th, 2004

  • Most basic operations implemented including:
    • Control over stachastic durations.
    • Order of Entry by Prime Complexity
    • Chebyshev Waveshaping oscillator with control over "Spectral Complexity"
    • Text display of status.
    • Graphical display of waveform.
    • File save and load.
    • Start and Stop
    • Setting of fundamental by frequency.
  • The following features are not implemented:
    • Gradual fade between harmonics.
    • Variation in section durations.
    User selection of Order of Entry algorithms.
    Pause and Resume
    • Setting of fundamental by MIDI pitch.
